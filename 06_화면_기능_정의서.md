## 3. 비회원 진단 (SCR-W-ONB-002) — 핵심 전환 지점

### 3.1 플로우
1. 랜딩 주 CTA 클릭 → 진단 도입 화면 (30초 설명, "5분이면 됩니다")
2. 15문항 적응형 시작 — 객관식·코드 읽기 혼합
3. 14번째 문항까지 답하면 상단에 "거의 다 왔어요 — 로그인하면 결과를 저장해 드릴게요" 배너
4. 15번째 완료 → 결과 직전 로그인 게이트 (OAuth2)

### 3.2 문항 UI
```text
┌──────────────────────────────────────────────────────────┐
│  문항 7/15   ⏱ 3분 남음                                  │
│                                                          │
│  다음 코드의 문제점은?                                    │
│                                                          │
│  @Transactional                                          │
│  public List<Order> findAll() {                          │
│    return repo.findAll();  // Orders has lazy items       │
│  }  // OSIV off, orders returned to view                 │
│                                                          │
│  ○ 성능만 살짝 나빠짐                                     │
│  ○ LazyInitializationException 발생 가능                  │
│  ○ 트랜잭션 안에서만 문제                                  │
│  ○ 문제 없음                                              │
└──────────────────────────────────────────────────────────┘
```

### 3.3 기능
| ID | 기능 | 설명 |
|----|------|------|
| ONB-02-01 | 적응형 난이도 | 정답 시 +0.1, 오답 시 -0.05 |
| ONB-02-02 | 코드 읽기 혼합 | 객관식 70% + 코드 리뷰 30% |
| ONB-02-03 | Bloom 분포 | Remember~Create 골고루 |
| ONB-02-04 | 중단 저장 | 30분 이탈 시 Redis에 상태 유지 |
| ONB-02-05 | 결과 게이팅 | 15번째 직후 로그인 강제 |

```text
3/4  12주 로드맵을 설계하는 중...
```

```text
│  └──────────────────────────────────────────────────────┘    │
│                                                              │
│  [클릭 시 신고 상세]                                          │

│  │ 원문 보기 │ 신고 사유(3건)           │                     │

└──────────────────────────────────────────────────────────────┘
```

## 14. 에러 상태 화면 정의

모든 에러 상태는 사용자에게 **원인 + 다음 행동**을 명확히 제공한다.

| 에러 상황 | 화면 표시 | 사용자 액션 | 자동 동작 |
|----------|----------|-----------|----------|
| **Sandbox 타임아웃** (30초 초과) | "실행 시간 초과 (30초)" + 코드 보존 | [코드 수정 후 재실행] [힌트 보기] | 세션 자동 종료, 실행 횟수 차감하지 않음 |
| **AI 코드 리뷰 실패** | "AI 리뷰 생성에 실패했어요" + 재시도 안내 | [다시 요청] [AI 멘토에게 질문] | 3회 재시도 후 실패 시 표시. Kill-switch 활성화 시 별도 안내 |
| **GitHub 프로필 수집 실패** | "GitHub 정보를 가져오지 못했어요" + 권한 확인 안내 | [권한 다시 확인] [GitHub 없이 진행] | Rate limit인 경우 "N분 후 자동 재시도" 표시 |
| **학습 경로 생성 오류** | "경로 생성 중 문제가 발생했어요" + SSE 진행률 롤백 | [다시 생성] [수동 트랙 선택] | 에러 로그 자동 수집, FinOps 대시보드 알림 |
| **네트워크 연결 끊김** | 상단 배너 "오프라인 — 연결 복구 시 자동 저장" | (자동) | 로컬 스토리지 캐시, 재연결 시 동기화 |
| **서버 500 에러** | "일시적 오류 — 잠시 후 다시 시도해 주세요" + trace_id | [다시 시도] [문의하기] | trace_id를 클립보드 복사 가능 |
