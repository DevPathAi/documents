# 15. 사용자 매뉴얼

> **문서 성격**: v1 목표 사용자 매뉴얼. 설명하는 기능 상당수(AI 리뷰 4~6초 응답, 월 한도 초과 처리, freeze 토큰 등)는 아직 부분 구현이거나 TARGET이다. 현재 구현 현황은 [42_전체_정합성_점검_2차](./42_전체_정합성_점검_2차.md)를 기준으로 확인한다.
>
> **대상**: 학습자 (Web + Mobile)
> **범위**: DevPath AI 주요 기능 사용법

---

## 1. 시작하기

### 1.1 가입
1. 랜딩 페이지 → [무료로 학습 경로 받기]
2. **가입 없이** 진단 테스트 15문항을 먼저 체험할 수 있습니다 (약 5분)
3. 결과를 받기 직전 OAuth 로그인 (GitHub / Google / 카카오)
4. **GitHub 로그인 권장**: 공개 리포 활동을 분석해 맞춤 추천 품질이 올라갑니다

### 1.2 진단 테스트
- 15문항 적응형. 처음은 쉬운 문제, 맞힐수록 어려워집니다
- 코드 읽기 문제 (`이 코드의 문제는?`) 섞여 나옵니다
- 중단해도 30분 내 이어서 가능
- Bloom's Taxonomy (기억→이해→적용→분석→평가→창조) 분포로 깊이 측정

### 1.3 학습 시간 약속
- 주당 시간과 선호 시간대를 설정하세요
- 알림과 목표 속도 계산에 사용됩니다 (정해진 시간 외 푸시 안 보냅니다)

## 2. 학습 경로 (1st Aha)

### 2.1 3개월 로드맵
로그인 직후 3~8초 로딩 후 **당신만의 12주 로드맵**이 생성됩니다.

- 📊 **현재 수준** (진단 기반: Junior 초반 / 중반 / Mid 등)
- 💡 **AI 멘토 코멘트**: "JPA 기초는 탄탄한데 N+1 심화가 필요해요"
- 🗓 **12주 타임라인** + **이번 주 과제 3개**
- 🎯 **완료 시 역량**: "3주 후 N+1 케이스 3가지 자동 감지"

단일 CTA: **"오늘의 학습 시작"**

### 2.2 경로 재생성
상단 메뉴 "경로 다시 만들기" → 이유 입력 → 새 경로 생성. 이전 경로는 히스토리에 보관.

## 3. 콘텐츠 학습

### 3.1 뷰어 구성
- Markdown 본문 + 코드 블록
- 코드 블록에 [▶ 실행해보기] — 그 자리에서 Sandbox 팝오버로 실행

### 3.2 진척도 자동 추적
80% 이상 읽고 체류하면 자동으로 COMPLETED 처리됩니다.

---

## 4. Sandbox 실습

### 4.1 실행 환경
- 격리된 Docker 컨테이너 (Java 21 / Node 20 / Python 3.12)
- CPU 1 vCPU · 메모리 512MB · 실행 시간 30초
- 네트워크 접속 차단 (보안)

### 4.2 사용 방법
1. 콘텐츠 내 실습 과제 → [코드 에디터 열기]
2. Monaco 에디터 (VSCode UX) — 키바인딩, 자동완성 동일
3. [실행] — 로그 실시간 스트리밍
4. [제출] — 테스트 케이스 통과 여부 + **AI 코드 리뷰** 자동 생성

### 4.3 단축키
- `Ctrl/Cmd+Enter`: 실행
- `Ctrl/Cmd+S`: 자동 저장 (3초마다 자동)
- `F1`: 커맨드 팔레트

## 5. AI 코드 리뷰

제출 후 4~6초 내 결과:
- 📌 요약 / 💪 잘한 점 / 💡 개선 제안 (라인 번호 포함) / 🔒 보안 이슈
- 👍👎 피드백으로 리뷰 품질을 함께 높일 수 있습니다

예시:
> ⚠️ **[MEDIUM] line 42** — LazyInitializationException 가능성. OSIV 끄고 `@EntityGraph` 또는 fetch join 권장.

## 6. AI 멘토

### 6.1 일반 ChatGPT와 다른 점
멘토는 **지금 당신이 뭐 하고 있는지** 압니다:
- 진행 중인 콘텐츠
- 최근 5개 Sandbox 제출 (실패 포함)
- 실패한 테스트 이름

그래서 이렇게 답합니다:
> "당신이 방금 실패한 `testFetchJoin`을 보면, `findAll` 호출 시 items가 Lazy라 N+1이 생깁니다..."

### 6.2 사용 방법
1. 콘텐츠/에디터에서 [질문하기]
2. 자연어 질문
3. 응답 하단 "참고" 섹션에 근거 링크 표시 (콘텐츠/내 Sandbox)

## 11. FAQ

**Q. GitHub로 로그인하면 내 코드도 올라가나요?**
아니요. 공개 리포의 메타데이터(언어/커밋 활동/토픽)만 가져옵니다. 코드 내용은 접근하지 않습니다.

**Q. Sandbox에서 `Socket`이 안 열려요**
의도된 동작입니다. 보안을 위해 아웃바운드 네트워크가 차단됩니다. 필요한 데이터는 로컬 mock으로 주어집니다.

**Q. AI 코드 리뷰가 틀렸어요**
응답 하단 👎로 피드백 주세요. 프롬프트 개선에 사용됩니다. 강사급 정확도는 v2.0 목표.

**Q. 학습 경로를 바꾸고 싶어요**
상단 메뉴 "경로 다시 만들기" + 이유 입력. 이전 경로도 보관됩니다.

**Q. 한도 초과는 어떻게 되나요?**
월 한도(멘토 20 / Sandbox 100) 초과 시 해당 기능만 다음 달까지 일시 중단됩니다. 콘텐츠 열람과 경로는 계속 가능합니다.
