# DevPath AI — 한국 백엔드 개발자 커리어 맞춤 AI 학습 플랫폼

> **"내 수준에 맞는 다음 단계를 AI가 안내하는 최초의 개발자 교육"**

한국 백엔드 개발자 커리어를 타겟으로, **AI 개인화 학습 경로**와 **실습 기반 AI 코드 리뷰**, **학습 맥락 연결 커뮤니티**를 결합한 학습 플랫폼입니다.

가입 후 10분 안에 두 번의 Aha Moment를 목표로 설계되었습니다:
1. **AI 개인화 학습 경로** — GitHub 활동 + 진단 결과 기반 3개월 로드맵
2. **AI 코드 리뷰** — 첫 실습 과제 해결 후 실무급 AI 피드백 수신

## 🔑 핵심 기술 결정

- **Backend**: Spring Boot 4 + Java 21 (Virtual Threads) + OAuth2 + Outbox
- **Data**: PostgreSQL (SSOT) + pgvector (콘텐츠·GitHub 임베딩) + Redis + Kafka + Elasticsearch
- **AI**: dev Ollama(`nomic-embed-text`, `qwen2.5:7b`) / 운영 목표 Claude 등 provider 교체
- **Sandbox**: Docker + **gVisor (runsc)** + 네트워크 차단 + 30초 limit
- **Web**: Flutter Web (Dart) + Riverpod 3 + go_router + dio + Monaco Editor(Sandbox)
- **Mobile**: Flutter 3.x + Riverpod + drift
