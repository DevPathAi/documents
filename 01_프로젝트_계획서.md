## 1. 프로젝트 개요

### 1.1 한 줄 요약

**"당신의 GitHub과 실력을 아는 유일한 한국 개발자 학습 플랫폼"** — 한국 백엔드 개발자 커리어를 타겟으로, AI 개인화 학습 경로와 실무급 AI 코드 리뷰를 결합한 학습 플랫폼.

### 1.2 배경

기존 개발자 교육(인프런, 패스트캠퍼스, Udemy)은 강의 일방향 소비 모델이다. 학습자 개인의 수준·목표·커리어 단계는 고려되지 않고, 실습 환경과 실무급 피드백은 분리되어 있다. 한국 시장에는 **"내 GitHub 활동과 현재 실력에 맞춰 다음 3개월을 짜주는 플랫폼"**이 존재하지 않는다.

DevPath AI는 세 가지 질문에 답한다:
1. 지금 내 실력에서 **다음 단계로 가려면 무엇을 배워야 하는가**? → AI 개인화 학습 경로
2. 내 코드에 대해 **실무급 피드백을 받을 수 있는가**? → Sandbox + AI 코드 리뷰
3. 내 학습 맥락을 아는 **개발자 커뮤니티**가 있는가? → 학습 맥락 자동 첨부 Q&A

> v2.0 이후 재검토 예정: AR 아키텍처 체험은 [22_AR_아키텍처_체험_참고설계.md](./22_AR_아키텍처_체험_참고설계.md)에서 참고 설계로 보존.

### 1.3 핵심 전략: 두 번의 Aha Moment

사용자 여정은 **진입 → Aha Moment → 습관화** 3단계로 설계한다. DevPath AI의 차별성은 **가입 후 1시간 안에 두 번의 Aha Moment**를 제공하는 것이다.

| # | Aha Moment | 시점 | 경험 |
|---|-----------|------|------|
| 1st | AI 개인화 학습 경로 | 가입 후 5~8분 | "내 GitHub과 진단 결과로 3개월 로드맵이 생성됨" |
| 2nd | 첫 과제 해결 + 실무급 AI 코드 리뷰 | 첫 학습 세션 내 30~60분 | "`line 42: LazyInitializationException 가능성` 수준의 현직자급 리뷰" |

### 1.4 대상 사용자

| 페르소나 | 상태 | 핵심 니즈 |
|---------|------|-----------|
| 지수 (24, 부트캠프 3개월차) | 취업 준비 | "지금 뭘 공부해야 할지 모르겠다" |
| 준호 (27, 주니어 3년차) | 이직 | "실력이 정체됐는데 원인을 모르겠다" |
| 수민 (31, 시니어 7년차) | 실무 역량 강화 | "내 코드에 실무급 피드백과 팀에 공유할 학습 자산이 필요하다" |

공통 요구: **한국어 콘텐츠**, **한국 기업 기술 스택 기준**, **실무급 피드백**.

### 1.5 핵심 목표

| # | 목표 | 설명 |
|---|------|------|
| ① | AI 개인화 학습 경로 | GitHub 활동 + 진단 테스트 기반 3개월 로드맵 |
| ② | Sandbox 실습 | Docker 격리 환경에서 Monaco 에디터로 코드 작성·실행 |
| ③ | AI 코드 리뷰 | Claude API 기반 실무급 코드 리뷰 (LazyInitializationException 수준) |
| ④ | AI 멘토 | 학습 컨텍스트 인식 Q&A (일반 ChatGPT 대비 차별점) |
| ⑤ | **커뮤니티 (학습과 동등한 축)** | **5개 게시판 + 스택오버플로 평판 + 학습 맥락 자동 첨부 + AI 시드 답변** |
| ⑥ | 리텐션 루프 | 스트릭 + 주간 리포트 + 정체 탐지 AI |

### 1.6 범위 (v1.0 MVP)

**포함:**
- OAuth2 가입 (GitHub/Google/카카오) + GitHub 활동 수집
- 15문항 적응형 진단 테스트 (Bloom's Taxonomy 태깅)
- AI 개인화 학습 경로 생성 (Claude + pgvector)
- 인터랙티브 콘텐츠 (Markdown + 실행 가능 코드 블록)
- Sandbox Runner (Docker + Monaco 에디터)
- AI 코드 리뷰 (Claude)
- AI 멘토 채팅 (컨텍스트 인식)
- **커뮤니티**: Q&A / 자유 / 프로젝트 공유 게시판 + 평판·태그별 평판 + Bronze 배지 9종 + AI 시드 답변 + 학습 맥락 자동 첨부 + AI 모더레이션
- 대시보드 (스트릭·진척률·배지)

**제외(v2.0 이후):**
- 1:1 인간 멘토 매칭
- **AR 아키텍처 체험** (MSA·자료구조 3D 씬) — 참고 설계는 [22_AR_아키텍처_체험_참고설계.md](./22_AR_아키텍처_체험_참고설계.md)
- 한국 외 언어 지원
- 스터디 모집 게시판, 수료자 라운지, 현상금 시스템
- 신뢰 사용자 권한(평판 10,000+), Silver/Gold 배지, 태그 위키

## 3. 사용자 여정 (End-to-End)

```text
Step 0. 랜딩 (히어로 데모 영상)
  ↓ "무료로 학습 경로 받기"
Step 1. OAuth2 가입 (GitHub 우선)
  ↓ GitHub API 백그라운드 수집
Step 2. 온보딩 (5분): 목표 → 트랙 → 진단 15문항 → 학습 시간
  ↓
Step 3. 🎉 1st Aha — 개인화 학습 경로 수신 (3~8초 로딩)
  ↓ "오늘의 학습 시작"
Step 4. 첫 학습 세션 (Markdown + 코드 블록)
  ↓ 실습 과제 열기
Step 5. Sandbox 실습 (Docker + Monaco) → 테스트 통과
  ↓ 비동기 AI 코드 리뷰 수신
Step 6. 🎉 2nd Aha — 실무급 AI 코드 리뷰 수신 + 커뮤니티 공유 제안
  ↓
Step 7. 커뮤니티 Q&A / AI 멘토
  ↓
Step 8. 대시보드 (스트릭·진척·배지)
  ↓ 리텐션 트리거 3종 (지속 루프)
```

상세 화면 흐름은 [05_화면_흐름_시퀀스_다이어그램.md](./05_화면_흐름_시퀀스_다이어그램.md), 스토리보드는 [08_스토리_보드.md](./08_스토리_보드.md).

## 4. 개발 일정 (총 24주)

| Phase | 기간 | 범위 | 마일스톤 |
|-------|------|------|----------|
| Phase 1 | Week 1-4 | OAuth2 + 기본 스키마 + GitHub 연동 + 커뮤니티 스키마 선반영 | M1: 가입 플로우 |
| Phase 2 | Week 5-8 | 진단 테스트 + 개인화 학습 경로 (Claude + pgvector) + Q&A 기본 | **M2: 1st Aha** |
| Phase 3 | Week 9-12 | 콘텐츠 뷰어 + Sandbox Runner + AI 코드 리뷰 + AI 시드 답변 | **M3: 2nd Aha** (실무급 리뷰) |
| Phase 4 | Week 13-16 | 모바일 앱 기본 + 자유/프로젝트 게시판 + 사용자 프로필 + UX 안정화 | M4: 모바일 + 커뮤니티 확장 |
| Phase 5 | Week 17-20 | AI 멘토 + 학습 맥락 연결 + 평판·배지 + 리텐션 루프 | M5: 습관화 |
| Phase 6 | Week 21-24 | 모더레이션 + FinOps + 부하/보안/QA + Founding Contributors 활성화 | M6: v1.0 릴리즈 |
