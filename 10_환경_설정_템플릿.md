# 10. 환경 설정 템플릿

> **대상**: 로컬 / CI / staging / production
> **구성 요소**: Docker Compose, `.env`, Spring profiles, Flutter flavor, IDE
>
> **현재 기준(2026-06-19)**: 로컬 인프라는 `devpath-shared/docker-compose.yml`의 PostgreSQL 17, pgvector, Redis, Kafka KRaft, Elasticsearch를 사용한다. 프론트엔드는 Flutter/melos, dev AI gateway는 Ollama 기준이다.

---

## 1. 환경 매트릭스

| 환경 | 용도 | DB | LLM | Sandbox |
|------|------|-----|-----|---------|
| `dev` | 로컬 | Docker PostgreSQL/pgvector/Redis/Kafka/ES | Ollama 또는 Mock | 모킹 runner, 후속 Docker pool |
| `ci` | 자동 테스트 | PostgreSQL service container 또는 Testcontainers | Mock | 모킹 runner |
| `staging` | QA/UAT | Managed PostgreSQL | Claude 등 staging provider | gVisor 풀 |
| `prod` | 운영 | Managed PostgreSQL HA | Claude 등 운영 provider | HPA 풀 |

## 2. Docker Compose (로컬 dev)

**파일**: `devpath-shared/docker-compose.yml`

```yaml
name: devpath-local
services:
  postgres:
    image: postgres:17-alpine
    ports: ["5432:5432"]
    volumes: [postgres-data:/var/lib/postgresql/data]
    environment:
      POSTGRES_USER: devpath
      POSTGRES_PASSWORD: localdev
      POSTGRES_DB: devpath

  postgres-vector:
    image: pgvector/pgvector:pg17
    ports: ["5433:5432"]
    volumes: [pgvector-data:/var/lib/postgresql/data]
    environment:
      POSTGRES_USER: devpath
      POSTGRES_PASSWORD: localdev
      POSTGRES_DB: devpath_vector

  redis:
    image: redis:7-alpine
    ports: ["6379:6379"]

  kafka:
    image: apache/kafka:3.9.1
    ports: ["9092:9092"]

  elasticsearch:
    image: docker.elastic.co/elasticsearch/elasticsearch:8.19.4
    ports: ["9200:9200"]
    environment:
      discovery.type: single-node
      xpack.security.enabled: "false"

volumes:
  postgres-data:
  pgvector-data:
```

## 4. Spring Profiles

### 4.1 `application.yml` (공통)
```yaml
spring:
  application:
    name: devpath-api
  security:
    oauth2:
      client:
        registration:
          github:
            client-id: ${GITHUB_OAUTH_CLIENT_ID}
            client-secret: ${GITHUB_OAUTH_CLIENT_SECRET}
            scope: [read:user, public_repo]
          google:
            client-id: ${GOOGLE_OAUTH_CLIENT_ID}
            client-secret: ${GOOGLE_OAUTH_CLIENT_SECRET}
            scope: [email, profile]
          kakao:
            client-id: ${KAKAO_OAUTH_CLIENT_ID}
            client-secret: ${KAKAO_OAUTH_CLIENT_SECRET}
            authorization-grant-type: authorization_code
  datasource:
    url: ${DB_URL}
  jpa:
    hibernate:
      ddl-auto: validate
  flyway:
    enabled: true

devpath:
  ai:
    ollama:
      base-url: ${OLLAMA_BASE_URL:http://localhost:11434}
      embed-model: ${OLLAMA_EMBED_MODEL:nomic-embed-text}
      chat-model: ${OLLAMA_CHAT_MODEL:qwen2.5:7b}
    provider:
      production: ${AI_PROVIDER:claude}
  sandbox:
    runtime: ${SANDBOX_RUNTIME}
    cpu: ${SANDBOX_CPU_LIMIT}
    memory-mb: ${SANDBOX_MEMORY_LIMIT_MB}
    timeout-sec: ${SANDBOX_TIMEOUT_SEC}
  limits:
    monthly:
      mentor: ${DEFAULT_MONTHLY_MENTOR_LIMIT}
      sandbox: ${DEFAULT_MONTHLY_SANDBOX_LIMIT}
    daily:
      sandbox: ${DEFAULT_DAILY_SANDBOX_LIMIT}
  features:
    ai-mentor-context-injection: true
    guest-assessment: true
```

## 6. Mobile (Flutter)

### 6.1 Flavor 구성

| Flavor | API | 번들 ID |
|--------|-----|--------|
| dev | ngrok / LAN IP | `ai.devpath.app.dev` |
| staging | staging API | `ai.devpath.app.staging` |
| prod | prod API | `ai.devpath.app` |

### 6.2 `lib/core/config.dart`
```dart
class Config {
  static const apiBaseUrl = String.fromEnvironment('API_BASE_URL');
  static const sentryDsn = String.fromEnvironment('SENTRY_DSN');
}
```

실행:
```bash
flutter run --flavor dev \
  --dart-define=API_BASE_URL=http://10.0.2.2:8080/api/v1
```

## 8. 최초 셋업 명령

```bash
# 1. 클론
git clone https://github.com/Public-Project-Area-Oragans/devpath-ai.git
cd devpath-ai

# 2. 환경 변수
cp .env.example .env
# OAuth Client ID/Secret, 운영 LLM provider key가 필요한 경우 기입

# 3. 인프라 기동
cd devpath-shared
docker compose up -d postgres postgres-vector redis kafka elasticsearch

# 4. DB 마이그레이션
./gradlew flywayMigrate

# 5. Backend 예시
cd ../devpath-platform-svc && ./gradlew bootRun

# 6. Frontend (Flutter 모노레포: 사용자/관리자 웹)
cd ../devpath-frontend && melos bootstrap
cd apps/web && flutter run -d chrome

# 7. Mobile (실기기 또는 에뮬레이터)
cd ../mobile && flutter pub get && flutter run
```
