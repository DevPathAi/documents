# 10. 환경 설정 템플릿

> **대상**: 로컬 / CI / staging / production
> **구성 요소**: Docker Compose, `.env`, Spring profiles, Flutter flavor, IDE

---

## 1. 환경 매트릭스

| 환경 | 용도 | DB | LLM | Sandbox |
|------|------|-----|-----|---------|
| `dev` | 로컬 | Docker PostgreSQL/Redis/pg | Mock or Claude | Docker pool (작음) |
| `ci` | 자동 테스트 | Testcontainers | Mock | 모킹 runner |
| `staging` | QA/UAT | Managed DB | Claude staging key | gVisor 풀 |
| `prod` | 운영 | Managed DB HA | Claude + OpenAI fallback | HPA 풀 |

## 2. Docker Compose (로컬 dev)

**파일**: `infra/docker/docker-compose.yml`

```yaml
version: '3.8'
services:
  api:
    build: ./devpath-api
    ports: ["8080:8080"]
    depends_on: [postgres, redis, kafka, elasticsearch, pgvector]
    environment:
      SPRING_PROFILES_ACTIVE: dev
      CLAUDE_API_KEY: ${CLAUDE_API_KEY}
      GITHUB_OAUTH_CLIENT_ID: ${GITHUB_OAUTH_CLIENT_ID}
      GITHUB_OAUTH_CLIENT_SECRET: ${GITHUB_OAUTH_CLIENT_SECRET}
      GOOGLE_OAUTH_CLIENT_ID: ${GOOGLE_OAUTH_CLIENT_ID}
      KAKAO_OAUTH_CLIENT_ID: ${KAKAO_OAUTH_CLIENT_ID}

  web:
    build: ./devpath-web
    ports: ["3000:3000"]
    environment:
      VITE_API_BASE_URL: http://localhost:8080/api/v1

  postgres:
    image: postgres:17-alpine
    ports: ["5432:5432"]
    volumes: [postgres-data:/var/lib/postgresql/data]
    environment:
      POSTGRES_PASSWORD: ${DB_ROOT_PASSWORD}
      POSTGRES_DB: devpath

  redis:
    image: redis:7-alpine
    ports: ["6379:6379"]

  zookeeper:
    image: confluentinc/cp-zookeeper:7.5.0
    environment:
      ZOOKEEPER_CLIENT_PORT: 2181

  kafka:
    image: confluentinc/cp-kafka:7.5.0
    ports: ["9092:9092"]
    depends_on: [zookeeper]

  pgvector:
    image: pgvector/pgvector:pg16
    ports: ["5433:5432"]
    volumes: [pgvector-data:/var/lib/postgresql/data]
    environment:
      POSTGRES_PASSWORD: ${PG_PASSWORD}
      POSTGRES_DB: devpath_vec

  elasticsearch:
    image: elasticsearch:8.12.0
    ports: ["9200:9200"]
    environment:
      discovery.type: single-node
      xpack.security.enabled: "false"

  minio:
    image: minio/minio
    ports: ["9000:9000", "9001:9001"]
    command: server /data --console-address ":9001"
    # 로컬에서 사용자 업로드 파일 대체 스토리지

  zipkin:
    image: openzipkin/zipkin
    ports: ["9411:9411"]

  prometheus:
    image: prom/prometheus
    ports: ["9090:9090"]
    volumes:
      - ./infra/prometheus/prometheus.yml:/etc/prometheus/prometheus.yml

  grafana:
    image: grafana/grafana
    ports: ["3001:3000"]

  sandbox-runner:
    build: ./devpath-sandbox-runner
    privileged: true   # runsc 사용 위해 (로컬만)
    environment:
      RUNNER_POOL_SIZE: 5
      RUNNER_RUNTIME: runsc
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock

  ai-review-worker:
    build: ./devpath-ai-review-worker
    depends_on: [kafka]
    environment:
      KAFKA_BOOTSTRAP: kafka:9092
      CLAUDE_API_KEY: ${CLAUDE_API_KEY}

volumes:
  postgres-data:
  pgvector-data:
```

## 4. Spring Profiles

### 4.1 `application.yml` (공통)
```yaml
spring:
  application:
    name: devpath-api
  security:
    oauth2:
      client:
        registration:
          github:
            client-id: ${GITHUB_OAUTH_CLIENT_ID}
            client-secret: ${GITHUB_OAUTH_CLIENT_SECRET}
            scope: [read:user, public_repo]
          google:
            client-id: ${GOOGLE_OAUTH_CLIENT_ID}
            client-secret: ${GOOGLE_OAUTH_CLIENT_SECRET}
            scope: [email, profile]
          kakao:
            client-id: ${KAKAO_OAUTH_CLIENT_ID}
            client-secret: ${KAKAO_OAUTH_CLIENT_SECRET}
            authorization-grant-type: authorization_code
  datasource:
    url: ${DB_URL}
  jpa:
    hibernate:
      ddl-auto: validate
  flyway:
    enabled: true

devpath:
  ai:
    claude:
      api-key: ${CLAUDE_API_KEY}
      model: ${CLAUDE_MODEL:claude-sonnet-4-6}        # 코드 리뷰, 학습 경로 생성
      model-light: ${CLAUDE_MODEL_LIGHT:claude-haiku-4-5}  # 멘토 경량 응답, AI 시드 답변
  sandbox:
    runtime: ${SANDBOX_RUNTIME}
    cpu: ${SANDBOX_CPU_LIMIT}
    memory-mb: ${SANDBOX_MEMORY_LIMIT_MB}
    timeout-sec: ${SANDBOX_TIMEOUT_SEC}
  limits:
    monthly:
      mentor: ${DEFAULT_MONTHLY_MENTOR_LIMIT}
      sandbox: ${DEFAULT_MONTHLY_SANDBOX_LIMIT}
    daily:
      sandbox: ${DEFAULT_DAILY_SANDBOX_LIMIT}
  features:
    ai-mentor-context-injection: true
    guest-assessment: true
```

## 6. Mobile (Flutter)

### 6.1 Flavor 구성

| Flavor | API | 번들 ID |
|--------|-----|--------|
| dev | ngrok / LAN IP | `ai.devpath.app.dev` |
| staging | staging API | `ai.devpath.app.staging` |
| prod | prod API | `ai.devpath.app` |

### 6.2 `lib/core/config.dart`
```dart
class Config {
  static const apiBaseUrl = String.fromEnvironment('API_BASE_URL');
  static const sentryDsn = String.fromEnvironment('SENTRY_DSN');
}
```

실행:
```bash
flutter run --flavor dev \
  --dart-define=API_BASE_URL=http://10.0.2.2:8080/api/v1
```

## 8. 최초 셋업 명령

```bash
# 1. 클론
git clone https://github.com/Public-Project-Area-Oragans/devpath-ai.git
cd devpath-ai

# 2. 환경 변수
cp .env.example .env
# OAuth Client ID/Secret, Claude API Key 발급 후 기입

# 3. 인프라 기동
docker compose -f infra/docker/docker-compose.yml up -d postgres redis kafka pgvector elasticsearch minio

# 4. DB 마이그레이션
./gradlew flywayMigrate

# 5. Backend
./gradlew :devpath-api:bootRun

# 6. Frontend (Flutter 모노레포: 사용자/관리자 웹)
cd devpath-frontend && melos bootstrap && (cd apps/web && flutter run -d chrome)

# 7. Mobile (실기기 또는 에뮬레이터)
cd devpath-frontend/apps/mobile && flutter pub get && flutter run
```
