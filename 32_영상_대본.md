# DevPath AI — 모두의 창업 첨부 영상 대본
## Vrew 제작용 · 3분 내외 · 차분한 전문가 톤 · 자동 스톡 영상 활용

> **영상 목적**: 모두의 창업 도전신청서 첨부용 영상. Q1~Q4 텍스트를 보완하여 **창업자의 진정성과 실행 의지**를 전달.
>
> **차별화 전략**: 숫자와 시장 분석은 텍스트에서 이미 전달됨. 영상은 "이 창업자가 진짜 이 문제를 풀 사람인가"에 대한 신뢰 형성에 집중.
>
> **분량**: 약 500자 (약 3분 발화 분량, 차분한 톤 기준)
>
> **작성일**: 2026-04-23

## 🎬 Vrew 음성 설정 추천

### 1순위 (가장 추천)

| 설정 항목 | 권장 값 |
|----------|---------|
| 음성 | **남성 · 성인 · 차분한 톤** (Vrew 기본 옵션: "재석", "지성", "경표" 계열) |
| 속도 | **1.0~1.05배** (차분한 톤 유지) |
| 발화 간격 | **보통** (너무 빠르면 무게감 상실) |

**이유**: 창업자 본인이 남성이라면 1인칭 서사와 자연스럽게 매칭됩니다. "내 이야기"라는 진정성이 유지됩니다.

### 대안 (필요 시)

| 경우 | 추천 |
|------|------|
| 창업자가 여성이거나 중성적 톤 선호 | 여성 · 차분 (예: "미소", "하연" 계열) |
| 더 공식적 무게 원함 | 남성 · 아나운서 톤 (예: "준영") |
| 부드러운 안내자 필요 | 여성 · 따뜻 (예: "소은") |

**절대 피해야 할 설정**: 발랄한 톤, 어린 목소리, 빠른 속도, AI 티가 많이 나는 기본 보이스.

## 📝 대본 본문 (Vrew에 그대로 복붙 가능)

### 섹션 1 — Hook & Personal Story (약 45초)

```
저는 2년제 대학에서 개발을 전공했습니다.

학원까지 다녔지만, 실무에 들어가자 턱없이 부족했습니다.

가장 답답했던 건 그 다음이었습니다.

뭘 더 배워야 하는지, 어디서부터 시작해야 하는지.

아무도 알려주지 않았습니다.

학원은 이미 수료했고, 온라인 강의는 내 수준을 몰랐습니다.

회사에도 가르쳐줄 사람이 없었습니다.
```

### 섹션 2 — Transition to Problem (약 35초)

```
결국 혼자 삽질하며 수많은 기술을 독학했습니다.

Spring Boot, React, Flutter, Docker, Kafka까지.

지금 돌이켜보면 누군가 한마디만 해줬어도.

"이 순서로 공부해." 그 한마디면.

최소 6개월은 아낄 수 있었습니다.
```

### 섹션 3 — Market & Bigger Problem (약 40초)

```
그리고 알게 되었습니다.

이건 저 혼자의 문제가 아니었습니다.

매년 25만 명이 개발 공부를 시작합니다.

하지만 컴퓨터공학과 취업률은 66%.

부트캠프 취업률도 비슷합니다.

이들에게는 공통적으로 사수가 없습니다.
```

### 섹션 4 — Solution & DevPath AI (약 45초)

```
DevPath AI는 이 빈자리를 AI로 채우는 플랫폼입니다.

먼저 실력을 진단합니다.

수준에 맞춘 학습 로드맵을 AI가 설계합니다.

막히면 질문합니다.

이때 학습 맥락이 자동으로 함께 전달됩니다.

답변자는 질문자의 수준을 알고 답합니다.

한국 플랫폼 최초의 기능입니다.
```

### 섹션 5 — Differentiation (약 25초)

```
핵심은 자동 보정입니다.

인강은 안 보면 끝입니다.

DevPath AI는 못 따라가면 경로가 바뀝니다.

학습자가 포기하기 전에, 시스템이 먼저 개입합니다.
```

### 섹션 6 — Why Me & Closing (약 40초)

```
저는 이 문제를 직접 겪은 창업자입니다.

그리고 이미 프로덕션급 시스템 두 개를 혼자 만들었습니다.

기술과 경험은 준비되어 있습니다.

월 14,900원. 개발자 한 명의 6개월을 아낄 수 있다면.

그것이 DevPath AI가 만들고 싶은 가치입니다.

함께 만들 기회를 주시기 바랍니다.
```

## ⚙️ Vrew 제작 워크플로우 (단계별)

### 1단계: Vrew 프로젝트 생성
1. Vrew 로그인 → 새 프로젝트
2. "AI 목소리로 영상 만들기" 선택
3. 언어: 한국어

### 2단계: 대본 입력
1. 위 **"통합본" 블록 전체를 복사**해서 Vrew 텍스트 입력창에 붙여넣기
2. Vrew가 문장 단위로 자동 컷 분할

### 3단계: 음성 설정
1. 상단 음성 설정 → **남성 · 차분한 톤** 선택
2. 속도: **1.0~1.05배**
3. "미리 듣기"로 어색한 부분 확인 → 필요 시 단어 간격 조정

### 4단계: 스톡 영상 확인
1. Vrew가 각 컷에 자동 매칭한 스톡 영상 재생
2. 어색한 컷은 위 **"스톡 영상 매칭 가이드"**의 키워드로 재검색
3. 원하는 영상 없으면 피치 덱 슬라이드 이미지로 교체

### 5단계: 자막 편집
1. 자막 폰트를 **나눔스퀘어 Bold** 또는 **Pretendard SemiBold**로 변경
2. 위 **"강조 지점"** 단어들을 굵게 또는 색상 변경

### 6단계: 배경음악 추가
1. Vrew 음악 라이브러리에서 **잔잔한 피아노** 검색
2. 볼륨 **15~20%**로 조정

### 7단계: 렌더링·다운로드
1. 전체 미리보기로 흐름 확인
2. **1080p MP4**로 내보내기
3. 파일명: `devpath-ai-intro-v1.mp4` 권장

### 8단계: 모두의 창업 업로드
1. 파일 크기 확인 (보통 100MB 이하 권장)
2. 도전신청서 첨부 파일 섹션에 업로드
3. 업로드 후 재생 테스트 필수
